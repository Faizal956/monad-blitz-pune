# 🎓 AI Campus Assistant

> An AI-powered student operating system built for the **Monad Hackathon**.

AI Campus Assistant helps students manage academics, skills, assignments, placements, and career growth through an intelligent AI agent while leveraging **Monad** for decentralized identity, verifiable achievements, and wallet-based reputation.

---

## 🚀 Features

### 📅 Smart Attendance
- Track attendance across subjects
- View attendance analytics
- Receive attendance alerts

### 📝 Assignment Planner
- AI-powered assignment scheduling
- Deadline reminders
- Priority-based task management

### 🤝 Club Matchmaker
- Discover clubs based on interests
- AI recommendations for extracurricular activities
- Skill-based matching

### 💼 Resume & Placement Assistant
- Resume improvement suggestions
- Placement preparation roadmap
- Interview guidance
- Internship recommendations

### ⭐ Skills & Reputation
- Five-dimensional skill analysis
- Technical Skills
- Business Skills
- Creativity
- Leadership
- Communication

### 🤖 AI Student Agent
The AI understands a student's:
- Skills
- Academic progress
- Career goals
- Interests
- Achievements

It provides personalized recommendations for learning, projects, internships, and hackathons.

---

# 🔗 Monad Integration

This project uses **Monad** to give students ownership of their academic achievements.

### Features

- 🔐 Wallet Authentication
- 👤 On-chain Student Identity
- 🏆 Achievement NFTs
- ⭐ Reputation Tracking
- 📜 Verifiable Academic Credentials

Instead of storing achievements only in a centralized database, important milestones can be verified and owned by students on Monad.

# 💡 Problem Statement

Students use multiple disconnected platforms for academics, placements, skill tracking, assignments, and career guidance.

AI Campus Assistant brings everything together into one intelligent platform while giving students ownership of their achievements through Monad.
